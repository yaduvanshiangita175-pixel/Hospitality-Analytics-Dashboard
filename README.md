# 🏨 Hospitality Analytics Dashboard

## 🔍 Objective
To analyze hotel business performance by evaluating booking trends, occupancy rates, cancellations, and revenue patterns.

---

## 🛠️ Tools Used
- Power BI  
- Tableau  
- Excel  
- SQL  

---

## 📈 Key Metrics
- Total Revenue  
- Occupancy %  
- Cancellation %  
- Booking Volume  
- Average Rating  

---

## ⚙️ Process
- Data Cleaning using SQL & Excel  
- Data Transformation  
- Dashboard Development  
- Trend Analysis & Visualization  

---

## 💡 Key Insights
- Identified cities contributing highest hotel revenue  
- Analyzed weekday vs weekend booking trends  
- Evaluated room class performance and occupancy behavior  
- Highlighted cancellation patterns affecting revenue
